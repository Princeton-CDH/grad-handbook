This static site hosts the graduate student handbook of the [Center for Digital Humanities at Princeton](https://cdh.princeton.edu/). It was made using the [hugo-book theme](https://github.com/alex-shpak/hugo-book/).

Images go in main directory's `assets/images` and use the figure.html shortcode like so:

`{{< figure src="images/gf_apps_all.png" alt="Alt text description." caption="Caption for the image." attr="Attribution" attrlink="link_to_resource" >}}`
